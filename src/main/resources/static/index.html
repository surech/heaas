HTTP Errors as a Service (HEaaS)

HEaaS generiert viele HTTP-Fehler über eine einfache Schnittstelle. Damit kannst du schnell überprüfen, wie sich deine Anwendung im Fehlerfall verhält.

HTTP Error Codes

Alle möglichen HTTP Error Code können einfach wie folgt generiert werden:

http://www.heaas.ch/errors/code/{HTTP_CODE}

Beispiele:

404 Not found
http://www.heaas.ch/errors/code/404
410 Gone
http://www.heaas.ch/errors/code/410
501 Not Implemented
http://www.heaas.ch/errors/code/501

Request Timeout

Mit dieser Schnittstelle kann man überprüfen, wie der Client reagiert, wenn der Server lange nicht reagiert. Per URL-Parameter kann man die Länge des Timeouts in Millisekunden sowie der resultierende HTTP-Code angeben.

http://www.heaas.ch/errors/timeout/{timeout}/{httpcode}

Bespiele:

http://www.heaas.ch/errors/timeout
Aufruf mit den Defaultwerten: 5 Minuten Timeout und anschliessend ein HTTP 408, Request Timeout
http://www.heaas.ch/errors/timeout/5000
Timeout von 5 Sekunden
http://www.heaas.ch/errors/timeout/5000/200
Timeout von 5 Sekunden und mit der Antwort HTTP 200, OK
